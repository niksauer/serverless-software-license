This project was bootstraped through https://github.com/bitjson/typescript-starter.

### Prerequisites

- Docker (Compose)
- `npm install -g truffle`
- `npm install -g yarn`

### Develop Library

1. `yarn install`
2. `yarn watch`

### Develop Smart-Contracts

1. `cd ethereum`
2. `docker-compose up`
3. `truffle build`
4. `truffle migrate`
